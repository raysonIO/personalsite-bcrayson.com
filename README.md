# AWS hosted personal site
This is a platform for my deep dive into cloud computing and understanding various tools. Kubernetes and AWS are my primary focus with possibly Linux and Azure more in the future.
I developed this site to be hosted on AWS through a static S3 bucket, served through cloudfront with Github actions for basic integration. The portfolio website itself will not have significant CI/CD as it is more of a visual than a full dev/prod implementation.
