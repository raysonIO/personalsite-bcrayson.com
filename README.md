# AWS hosted personal site
This is a platform for my deep dive into cloud computing and understanding various tools. Kubernetes and AWS are my primary focus with possibly Linux and Azure more in the future.
This site is hosted on AWS through a static S3 bucket, served through cloudfront with Github actions for integration. 
The portfolio website itself will not have significant CI/CD as it is more of a visual than a full dev/prod implementation. 
The convenience is the deployment of the site itself automatically rather than seperate tenants.
The portfolio site costs about 1 dollar to host.
